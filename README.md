# AMOLED Anime Theme — OneShot

An AMOLED (true black) Discord theme featuring Niko from the game **OneShot** by wiwiichan.

![Preview](previewUrl.jpg)

## Palette

| Role | Color |
|------|-------|
| Backgrounds | `#000000` (true black) |
| Accent / Brand | `#9324af` (plum purple) |
| Surfaces | `#16091b`, `#230d3b` |
| Text | `#FFFFFF` (white) |
| Muted text | `#7a5a8a`, `#a084b0` |

## Features

- True black backgrounds for AMOLED displays
- Purple/plum accent palette inspired by OneShot
- Custom background image (Niko) with 25% opacity and 0.5 blur
- Solar icon pack

## Installation

1. Copy the raw URL of `amoled-anime-theme.json`
2. Import it into your Discord client mod (Vencord, BetterDiscord, Replugged, etc.)
3. Enable the theme

## Files

- `amoled-anime-theme.json` — theme definition (Discord spec v2)
- `4099d53ea58fa7285f19c8f6b567f099.jpg` — background image asset
- `previewUrl.jpg` — theme preview
- `rem.jpg` — additional promo asset
